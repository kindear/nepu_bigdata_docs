# 🎓 大数据智能问答系统 — 课程教程

欢迎来到 **NEPU 大数据小学期** 课程教学内容站点。

## 📖 课程概述

本课程将带领你从零开始，构建一个完整的 **基于 RAG（检索增强生成）的大数据智能问答系统**。课程涵盖数据采集、存储、处理、知识库构建、Agent 工具调用到系统集成与测试的全流程。

## 🗺️ 技术栈

| 层级 | 技术选型 |
|------|----------|
| 数据采集 | Python Requests, BeautifulSoup, Selenium |
| 数据存储 | MySQL, MinIO (S3), JSONL/Parquet |
| 数据处理 | PySpark, Spark SQL |
| 向量数据库 | FAISS, Embedding 模型 |
| RAG 框架 | LangChain |
| 后端服务 | FastAPI |
| 前端界面 | Vue 3 |
| 大模型 | DeepSeek API |

## 📚 章节导航

1. **第一章 项目概述与技术路线** — 了解项目全貌与开发环境搭建
2. **第二章 数据采集** — 从 Web 获取数据并处理分页与动态内容
3. **第三章 数据存储与处理** — 设计存储方案并用 PySpark 清洗数据
4. **第四章 知识库与 RAG 问答** — 构建向量索引，实现智能检索问答
5. **第五章 Agent 与工具调用** — 让 AI 学会使用工具
6. **第六章 系统开发与集成** — 后端 API 与前端界面开发
7. **第七章 系统测试与项目验收** — Benchmark 测试与答辩准备
8. **附录** — 数据格式规范与代码评阅标准
